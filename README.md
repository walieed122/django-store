<h1>متجر إلكتروني باستخدام Django</h1>

<p>
هذا المستودع يحتوي على مشروع متجر إلكتروني تم تطويره باستخدام إطار العمل
Django، ضمن مسار تعلم إنشاء المتاجر الإلكترونية بعد دراسة الأساسيات من أكاديمية حسوب.
</p>

<h2>وصف المشروع</h2>
<p>
متجر إلكتروني متكامل يتيح عرض المنتجات، إدارتها، وتنفيذ عمليات الشراء
مع دعم أنظمة الدفع الإلكتروني، ويهدف إلى تطبيق المفاهيم العملية
لتطوير تطبيقات الويب باستخدام Django.
</p>

<h2>المميزات</h2>
<ul>
  <li>إدارة المنتجات (إضافة – تعديل – حذف)</li>
  <li>نظام سلة مشتريات</li>
  <li>إتمام الطلبات (Checkout)</li>
  <li>دعم الدفع الإلكتروني (قابل للتطوير)</li>
  <li>لوحة تحكم باستخدام Django Admin</li>
</ul>

<h2>المتطلبات</h2>
<ul>
  <li>Python >= 3.9</li>
  <li>pip</li>
  <li>virtualenv أو pipenv (اختياري)</li>
</ul>

<h2>طريقة التثبيت والتشغيل</h2>
<ul>
  <li>
    نسخ المستودع:
    <br>
    <code>git clone https://github.com/USERNAME/REPO_NAME.git</code>
  </li>
  <li>
    الانتقال إلى مجلد المشروع:
    <br>
    <code>cd REPO_NAME</code>
  </li>
  <li>
    تثبيت المتطلبات:
    <br>
    <code>pip install -r requirements.txt</code>
  </li>
  <li>
    تشغيل التهجير (Migrations):
    <br>
    <code>python manage.py migrate</code>
  </li>
  <li>
    تشغيل الخادم المحلي:
    <br>
    <code>python manage.py runserver</code>
  </li>
</ul>
